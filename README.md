# Personal Website Portfolio

A creative, retro-themed personal portfolio website featuring a dual-mode interface: an interactive terminal and modern GUI. Built with modern web technologies for a unique browsing experience.

## 🎨 Features

- **Terminal Mode** - Interactive command-line interface with file system navigation and portfolio info
- **GUI Mode** - Clean, modern interface with project showcase and contact information
- **Theme System** - 4 dynamic themes (default, pink, pinkDark, hacker)
- **Responsive Design** - Works seamlessly on desktop and mobile

## 🛠️ Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router
- shadcn-ui
- Bun
